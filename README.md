Um sistema simples em Java para cadastro e gerenciamento de Alunos e professores, utilizando princípios de POO, herança, polimorfismo e tratamento de exceções.


Aqui tem:
      Cadastro de pessoas (Aluno e Professor)
      Busca por nome
      Listagem de todos os cadastrados
      Exclusão de registros
      Uso de classes abstratas
      Sobrescrita do método toString()
      Uso de Enums para cargos
      Tratamento de erros com exceções personalizadas
      Sistema simples e funcional via console

Sobre exeção ( Trabalhei pouco nesse projeto ) 
      Salário não pode ser negativo
      Nome não pode ser vazio
      Matrícula não pode ser inválida
      Exceções personalizadas (DadosInvalidosException)

Para utilizar so da run no "main.java"
